Introduction

Welcome to the cloud deployment assignment. This project demonstrates the deployment of a website on AWS, highlighting core cloud concepts, infrastructure setup, and security configurations.

Setup

The infrastructure is built using AWS services, including EC2, S3, and VPC. The EC2 instance hosts this website, and security is enhanced through IAM roles and security groups.

Security Measures

Data at rest is encrypted using AES-256 for S3 buckets, while HTTPS ensures data in transit is secure. IAM roles enforce least privilege, and Multi-Factor Authentication (MFA) is enabled for user accounts.

Conclusion

This deployment demonstrates best practices in cloud computing, emphasizing scalability, security, and compliance with industry standards.
